Detailed Description

A conversion of pyptp2 and all that comes with it to C++.

libptp2 is nice, but appears to be tightly bound to ptpcam. There are a few other CHDK-specific programs to communicate with a camera through PTP, but all contain source code that is tightly integrated and difficult to read.

While this library should be able to communicate with any PTP camera through the PTPCamera interface, it's primary purpose is to allow easy communication with cameras running CHDK through CHDKCamera.

This library has two goals:

  1. Provide all functionality of pyptp2 through a C++ interface.
  2. Be easy to use, and well-documented.
