void PTP::PTPContainer::add_param ( uint32_t  param)

Add a parameter to a PTPContainer.

This function can add any uint32_t as a parameter, but is most useful for adding a member of CHDK_OPERATIONS or a parameter to that operation. But, the function is generic enough that any type of data could be added, so this can help create any generic PTP command.

Parameters
[in]paramThe parameter to be added

unsigned char * PTP::PTPContainer::pack ( )

Pack PTPContainer data into a byte stream for sending.

Packs the data currently stored in the PTPContainer into a array of unsigned characters which can be sent through USB, etc. to the device. The length of this data can be found with PTPContainer::get_length

Warning
Since this method malloc()s space for the packed data, the caller must make sure to avoid memory leaks by free()ing this data.
Returns
A pointer to the first unsigned character which makes up the data in the container.

PTPContainer::get_length

void PTP::PTPContainer::unpack ( unsigned char *  data)

Unpack data from a byte stream into a PTPContainer.

This function will overwrite any data currently stored in this PTPContainer with the new data from data. data is parsed for each part of the PTP message, and individual items are stored appropriately.

Warning
data must be at least 12 bytes in length, or this could segfault.
Parameters
[in]dataThe address of the first unsigned character of the new container data. Must be at least 12 bytes in length.
