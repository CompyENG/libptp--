char * PTP::CHDKCamera::_wait_for_script_return ( int  timeout)

Block until the currently running script returns a value.

This function will poll the camera every 50 ms for script messages. If a script is currently still running, it will continue to poll until all scripts are done running. All read messages are returned when all scripts are done running.

void PTP::CHDKCamera::get_live_view_data ( LVData *  data_out,
bool  liveview = true,

bool  palette = false 
)

Retrieve live view data from CHDK.

Returns selected frame buffers from CHDK in an LVData object. By default, only live view data is returned, but overlay and palette can optionally be returned, also. The LVData object can then be used to retrieve and manipulate the live view data.

Parameters
[out]data_outThe address of an LVData object which will be populated with the requested data
[in]liveviewTrue to return the live view frame buffer
[in]overlayTrue to return the overlay frame buffer
[in]paletteTrue to return the palette for the overlay
See Also
LVData, http://chdk.wikia.com/wiki/Frame_buffers
